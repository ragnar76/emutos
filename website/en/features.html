The EmuTOS operating system has the following features:

  • Complete Atari ST compatible BIOS interface (trap #13)
  • Complete Atari ST compatible XBIOS interface (trap #14)
  • Complete Atari ST compatible GEMDOS interface (trap #1)
  • Complete Atari ST compatible VDI interface (trap #2)
  • Complete Atari ST compatible AES interface (trap #2)
  • It is single-tasking
  • It is for a single user
  • It is very small (currently about 250 KB)
  • You can use two floppy disk drives
  • You can use up to 26 hard disk partitions
  • Its FAT filesystem is DOS compatible
  • Its console is VT52 compatible and additionally supports up to 16 colors
  • It supports all Atari ST video modes and VIDEL
  • It provides an internal minimalistic shell called EmuCON
  • It can be used to load MiNT, and later fVDI and XaAES

This is the currently supported Hardware:

  • CPU support for m68000, m68010, m68020, m68030, m68040, m68060, ColdFire V4e
  • FPU detected
  • Memory controller (both ST and Falcon)
  • Monitor type detection (mono or not)
  • WD 1772 Floppy disk controller (write track not tested)
  • DMA controller
  • MFP
  • YM-2149
  • ST/STe shifter
  • VIDEL (partially)
  • ACIAs, IKBD protocol
  • MegaST Real-Time Cloc
  • NVRAM (including Real-Time Clock)
  • DMA sound
